# Screenshots Folder

This folder contains screenshots of the AI Trading Platform for documentation purposes.

## Required Screenshots

Please add the following screenshots:

1. `home.png` - Landing page
2. `trading.png` - Trading terminal with candlestick chart
3. `dashboard.png` - Main dashboard with portfolio
4. `market.png` - Market overview page
5. `agents.png` - AI agents management page

## How to Take Screenshots

1. Open the application at `http://localhost:8080`
2. Login with demo credentials
3. Navigate to each page
4. Take a screenshot (Windows: Win + Shift + S)
5. Save as PNG file in this folder
